Einsatzplaner – Das Programm für Museumseisenbahnen

Kostenlos | Innovativ | Flexibel

Das neue Programm für Museumseisenbahnen

Mit diesem Programm ist es leichter als je zuvor Arbeitseinsätze zu Planen, Personal zu koordinieren und Reservierungen zu verwalten

Funktionen

  • Erstellen von Fahrtagen, Sonderzügen, Nikolauszügen und vielem mehr
  • Bearbeiten der jeweilgen Züge/Fahrtage
  • Verwalten der Reservierungen die zu einem Fahrtag eingehen
  • Automatische Verteilung der Sitzplätze (Kommt bald)
  • Anzeigen der bereits belegten Sitzplätze
  • Export Funktion

Erstellen von Fahrtagen

Mit einem Doppelklick auf einen Tag im Kalender können Sie einen Fahrtag mit dem gewüscnhten Datum erstellen.
Ebenso können Sie mit dem Klick auf den Button "Neuen Fahrtag anlegen" einen neuen Fahrtag mit dem gewählten Klick anlegen.

Mit einem einfachen Klick auf einen Tag im Kalender ist es möglich den jeweiligen Tag in der Liste anzuzeigen. Sodass man mit wenigen Klicks direkt prüfen kann, ob man an einem Tag schon einen Sonderzug hat und wenn nicht, dass man ihn direkt anlegen kann.

Bearbeiten von Fahrtagen

Wenn Sie einen Fahrtag in der Liste ausgewählt haben, können Sie ihn bearbeiten. Sie können im festen oberen Bereich die Rahmendaten für den Fahrtag einstellen. Die Wahl der Art des Fahrtages wird auch farblich in der Listenansicht dargestellt, sodass man auf den ersten Blick erkennen kann, ob es sich um einen normalen Museumstag handelt oder ob es zum Beispiel ein Nikolaus-Sonderzug ist.

Das Personal, das benötigt wird, können Sie in den Listen im Tab Personal eintragen. Auch kann hier vermekrt werden, ob noch weiteres Personal benötigt wird oder ob sich schon genügend Personal gemeldet hat.

Export Funktionen

Sobald Sie einen Fahrtag erstellt haben, haben Sie Zugriff auf die Export-Funktion.
Sie können im oberen Bereich einen Zeitraum wählen, in dem die Fahrtage liegen müssen, damit sie ausgegeben werden.
Auch können Sie die Art des Fahrtages wählen.

So können Sie zum Beispiel ganz einfach alle Sonderzüge eines Jahres ausgeben.

In der Liste sehen Sie eine Übersicht über die Fahrtage, die bei der Listenansicht ausgegeben werden. Sie können einen odere mehrere Fahrtage markieren, die Sie dann mit der Einzelansicht ausgeben können.
Ebenso können Sie zwischen einem Export als PDF-Datei wählen oder aber direkt drucken.

Kompatibilität

OS X

OS X wird ab Version 10.11 unterstützt. Das Programm sollte allerdings auch auf niedrigeren Systemen mit Intel-Prozessor lauffähig sein.

Windows

Windows wird ab Windows 7 unterstützt. Getestet wurde das Programm auf Windows 7 und Windows 10.

Linux und sonstige

Für alle, deren Betriebsystem oben nicht aufgeführt ist, gibt es noch die Möglichkeit, dass Sie sich das Programm selbst kompilieren. Das Programm ist komplett OpenSource und der Quellcode ist öffentlich einsehbar auf github.com . Den Link dazu finden Sie unten.

Sollten Sie wider Erwarten bei einem der unterstützten Systeme auf Probleme stoßen, wenden Sie sich bitte an den Support.
